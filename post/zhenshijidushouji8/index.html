真实缉毒手记：IT男贩毒漏洞百出，人赃并获

November 9, 2022 · 3 min · 投稿

那天是午饭时间，但饭太难吃了，我和老狗熊活活被老左逼成了逃饭，逃回宿舍去吃泡面。

但只要轮到你值日，你可以不吃饭，等别人吃完之后，你必须要洗全桌人的碗。

那天老狗熊值日，我们俩一起出来，老狗熊去洗碗，我去厕所。

路上我看到丁卓在远处操场上，一个过肩摔把另一个新兵摔了两百七十度，然后把人按在地上。

丁卓那按兵不动的姿势像是要掐死对方，我还以为跟谁决斗呢，赶紧跑过去，走近了才发现，原来在教新兵摔擒。

作为老兵的经验告诉我，这种传授武功的场面不能多呆。

我转身想走，但仍然迟了一步，被丁卓叫了回来，要我配合一下，说是要给新兵演示。简单说就是他要打给人家看，而我就是那个被打的。

我告诉丁卓可以打旁边的沙袋。

丁卓说沙袋还是活的好。

我闭眼感受了一下，确定自己现在的状态很容易被打爆，就是打成失禁状态，所以只能假装听不到跑了。

在我迅速远去的时候，单位通讯员却迎面跑来，然后以一个沙包的姿势朝丁卓跑去。

那我就不着急逃了，我往回跑准备看武打片。

他跑到丁卓面前，说了一下情况，简单说就是一男一女两个人出现在我们的区域，看上去行色匆匆的，像外地人，可疑。

我们集合，朝目的地出发，走得很快，目标位置在山背面，那一男一女也没走，就坐在原地休息。

女人背对我们坐在地上，男人悬浮在半空中，能看到侧脸，倒是不太像贩毒的人，没有罪犯气质，而且愁眉苦脸的，看上去倒像个文艺青年。人往石头上一站，有种郁树临风的气质。

然后我们就围上去，先是亮出自己的证件，然后检查了他们的证件，男人湖南的，女人四川的，俩人说是来徒步旅行的。

那我们当然不信，当地四十多度的气温，不带这么玩的。问他是做什么的，他好像说是软件工程师，我当时对这个职业完全无知，我的理解就是网管、修电脑的、或者高级一点做网页的。毕竟我自己当时也只是熟练操作QQ的水平。

这样的问话是有技巧的，比如他说他是软件工程师，你得问他软件行业的事情，才能确认他的身份有没有疑点。要是现在问，我可能会装作内行和他讨论下 Python 和 Java 的优劣，但那时候只知道问他会不会换电脑硬盘，或者使几台电脑联机，因为当时我们正在为不能连局域网打暗黑而苦恼。

得到的答案是都会，问话的确难不住他，还搞得我们很尴尬，于是我们开始搜男人，没有什么发现，行李也没啥发现。最后在女人的背包里发现一大包糖果。

丁卓说你这个糖果我们打开两颗看看行吧？正常人一般都会同意。认真点的人，会说你打开就得赔我钱。

但那个女人特别耿直地说：这个你们不能查。

问她为什么不能查，她也不说话。

丁卓说：你也知道我们是干什么的，既然你都这么说了，无论如何我都要查了。

女人想来夺包，被老狗熊拦住，丁卓开始随机剥糖果，大概就是每个颜色挑几颗出来，一共大概剥了五六颗，在紫色糖果里发现了海洛因。

俩人现场被上了铐子，带回单位的路上，丁卓开始问话，这时候发生一点小意外。

本来整个过程中，男人都没怎么说话，就是那种停智不前的样子，但上车之后突然变了个人，有了一种智咤风云的感觉。

他开始诉说自己的悲惨身世，说自己得了癌症，带货是为了治病。一边说还一边让女人给自己证明，可能是男人的即兴发挥，俩人没有提前串供，所以配合起来就有点磕巴。

男人说自己得癌症的时候，用下巴指了指女人，说：不信你们问她。

女人似乎 get 到了男人的点，而且非常紧张，说：对，我也得了癌症，我们俩都是为了治病。

俩人越说越惨，就差唱癌的路上只有我和你了。

后来审讯细节我没有参加，内勤做完笔录之后，我拿着路由器去找男人问了几个问题，从那以后就可以联机打暗黑了。

具体数量我记不清了，只记得货是两人从境外背回来的，第二天一早移交，数罪并罚，盖木欧瓦。
