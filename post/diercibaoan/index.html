11.第二次报案

November 27, 2022 · 1 min · 投稿 | Suggest Changes

「C 肽没什么特别作用，但是可以用来鉴别胰岛素的来源。人体内自行产生的胰岛素中含有一定比例的 C 肽，而注射用胰岛素不含 C 肽。田靓体内有大剂量胰岛素，但是 C 肽无明显升高，说明她体内的胰岛素是被注射进去的。」「大量胰岛素的摄入造成她的血糖急剧下降，进而导致死亡。她是被谋杀的！」冯可欣很激动。「现在下结论还为时过早，别忘了，田靓本来就是糖尿病患者，一直使用外源性胰岛素治疗，她体内有胰岛素沉积再正常不过。无法由此得出她是被谋杀的结论。」 光阴似箭，转眼过了一年。

省医大第一附属医院有人报案，说一名刚刚去世的病人的死因可疑。我和冯可欣接警后同车赶往现场。

已是黄昏时分，天气微凉，下班的人们行色匆匆地走在回家的路上。报案人坐在医院大门口的花池子旁等我们，丢了一地烟头。我们的车才停好，他就过来与我们握手，自我介绍说是死者的丈夫，名叫文良宇，楚原外国语学校的副校长。

文良宇看上去三十出头，中等身材，戴无框眼镜，眉清目秀，文质彬彬，一看就知是个知识分子。

冯可欣开门见山：「什么情况？」

文良宇说：「我妻子患有糖尿病，好几年了，一直在医大一院治疗。这次病情比较严重，医生建议住院观察。眼看一个星期了，医生说各项指标都已经降下来，而且很稳定，估计今明两天就能出院。昨晚我终于松了口气，回家睡了个好觉，留我妈在医院陪床。谁承想今天凌晨四点多钟，我突然接到我妈的电话，说我妻子……已经……不行了……」他按捺不住悲伤，掩面痛哭起来，搞得我鼻子也发酸。

冯可欣纳闷儿：「不是说病情已经好转，怎么突然又不行了？医生说是什么原因了吗？」

文良宇擤擤鼻涕，说：「医生说是血糖过低导致死亡。」

冯可欣越发迷糊：「你妻子患有糖尿病，怎么会血糖过低呢？这……说不通啊。」

我解释说：「在病理上，高血糖和低血糖并不矛盾，低血糖症是糖尿病治疗过程中最常见的并发症，糖尿病患者在饮食受到控制的情况下，如果能量得不到及时补充，就容易低血糖。注射胰岛素后没有按时进餐，或运动过量，或病情好转后未能及时减量，都是引起低血糖的原因。血糖过低时，患者会出现精神和意识障碍，甚至胡言乱语，再重者就会昏迷，危及生命。」

文良宇哽咽着说：「这些医生跟我解释过了，我当时心慌意乱，没怎么听明白。」

我说：「你根据什么认为你妻子不是病死的呢？」

文良宇想了好一会儿才说：「不根据什么，凭直觉。医生跟我说的话，解释的那些病因，我一句都没听进去。我就想，我妻子好端端的，各项指标都正常了，而且她有很强的求生欲，怎么会在睡梦中悄无声息地离我而去呢？这不可能，我绝不相信。」

我和冯可欣对视一眼，想这个可怜人因妻子突然去世魔怔了，没有一点根据就报警，多半要害我们白跑一趟了。

我问：「你妻子现在在哪里？」

文良宇刚刚止住的眼泪又喷涌而出：「医院催着腾出床位，我拦不住他们。上午人就被送去太平间了。」

「你怎么没尽早报案？」冯可欣问。

「他们不让，都拦着我，劝我，说些不痛不痒的话，我越听越烦。快到晚上了，我才抽出身来报警，他们还不知道呢。」

「如果想找出你妻子的确切死因，必须解剖尸体，你肯签同意书吗？」

「肯。」文良宇回答得很干脆。

田靓（文良宇妻子）的尸体被从冷柜里取出时，已呈冷冻状态，全身发青，表面覆盖着一层薄薄的霜。面部、手掌等裸露在外的部位已出现皱缩。

死者三十岁，身长一米六四，体重四十八公斤。黑长发。无生育史。体表有浅淡尸斑，无外伤。两手臂肘弯处及脐下各有一陈旧针眼。内脏器官无病变，无毒素沉积，无伤损破裂。在死者体内检验出大剂量的胰岛素，但是 C 肽无明显升高。

「最后一句话是什么意思？」冯可欣没明白。

「胰岛素是用来降血糖的，知道吧？」

「知道。」

「C 肽没什么特别作用，但是可以用来鉴别胰岛素的来源。人体内自行产生的胰岛素中含有一定比例的 C 肽，而注射用胰岛素不含 C 肽。田靓体内有大剂量胰岛素，但是 C 肽无明显升高，说明她体内的胰岛素是被注射进去的。」

「大量胰岛素的摄入造成她的血糖急剧下降，进而导致死亡。她是被谋杀的！」冯可欣很激动。

「现在下结论还为时过早，别忘了，田靓本来就是糖尿病患者，一直使用外源性胰岛素治疗，她体内有胰岛素沉积再正常不过。无法由此得出她是被谋杀的结论。」

冯可欣的性格偏多血质，情绪波动大，听我这么一说又泄了气：「那不是白忙活一场？」

「没白忙，至少证明田靓有百分之五十的可能是被注射过量胰岛素致死的。接下来能不能揭开盖子，就看刑侦的了。」

冯可欣想了想：「先回队里跟沈队汇报下再说吧。」

沈恕听后也很挠头——证据不足，没法立案。如果仅因家属报案和尸检结果就仓促立案，投入原本就非常紧张的警力进行侦查，最后若证实死者是自然死亡或死于医疗事故，那么对上对下都不好交代。如果不立案呢？毕竟是一条人命，而且死因存疑，不查明真相又于心不忍。

沈恕在办公室里兜了两个圈子，对冯可欣说：「先小范围调查。死者的婆婆当晚在医院陪床，她应该知道些情况，跟她碰一下。此外，死者生前的主治医生、护理团队，一个也别漏，挨个谈话。人不明不白地死在医院里，他们肯定想尽快了结，未必都说实话，你酌情处理，连哄带吓唬，让他们吐点干货出来。」

冯可欣做了个京剧里的盘袖动作：「得令。」

听到这里，肖蔷忍不住笑起来：「你们局里的刑警还挺有意思的。」

「苦中作乐呗。」我说，「刑警们任务重，压力大，作息没规律，都练就了自得其乐的本事。」

胡华华十分好奇地问：「田靓到底是不是被人害死的呢？最后调查清楚没有？」

肖蔷怼他：「肯定查清楚了，不然说这些干吗呢？」

胡华华不敢反讥领导，挠挠头，不好意思地笑笑。

田靓的主治医生赵天明，五十多岁，身高体胖，脸像橘子皮一样坑洼不平，说话声音洪亮，是省内知名的糖尿病专家。

「田靓死于低血糖，没有可疑之处。家属把你们都惊动了，浪费警力，浪费纳税人的钱，不应该，但是心情可以理解。」赵天明说，「以我从业三十多年的经验，我负责任地说，低血糖是糖尿病病人最常见的并发症，也是最大的死因之一。田靓这些日子受病痛折磨，胃口一直不好，我嘱咐她多吃东西，补充营养，她不肯听。」

赵天明痛心疾首地应和：「唉，她不肯听。」

冯可欣又去走访田靓去世那晚的值班护士李晴。

李晴二十一岁，护理专业本科毕业，参加工作还不到一年。冯可欣表明身份时她有点蒙，脸涨得通红：「警……警察？我摊上啥事了？」

冯可欣差点儿被她的反应逗笑：「警察又不吃人，用不着这么紧张。是关于田靓的事，她去世那天晚上是你值班吧？」

「是……是啊，咋了？」

「那天晚上你到田靓病房去了几次？」

「嗯……三次，七点钟交班时去了一次，看看病人的情况，十二点去给她换药，再去就是早上四点多钟，那时候人已经不行了。」

「那天夜里还有谁进过田靓的病房？」

「她丈夫，七点多离开的，说是回家休息。她丈夫从她住院起，每个晚上都来陪床，一个囫囵觉都没睡过，人相当好。就那天晚上回家睡觉去了，结果人就没了，他现在一定特后悔。再有就是田靓的婆婆，整晚都在病房里。别人……我没见到。」李晴说话很小心，字斟句酌的。

另一个重要证人是田靓的婆婆曹粤。她刚满六十岁，从教师岗上退下来不久。人很精神，满头乌丝，眼神明亮，吐字清楚，说话有条有理。

「田靓的父母都在外地，身体都不大好，听说女儿住院，心里着急却又来不了，我便主动申请陪床，替他们也替良宇分忧。田靓这孩子，嘴甜、勤快，又懂事，我把她当亲生女儿一样对待。她这一走，像是从我的心上剜掉了一块肉，我难过得一分钟也没合眼。」曹粤边说边擦眼泪。看得出她是真心难过，这样亲如母女的婆媳关系还是很难得的。

「除你之外，那天夜里有几个人进过田靓的病房？」

「赵天明来过，看了看田靓的血糖指标和用药情况，说一切平稳正常，和我聊了几句，就走了。护士李晴来过两次，每次停留十来分钟，她工作挺细心的。再有就是凌晨四点多钟的时候，田靓的情况不好，整个医疗团队都先后赶来。我当时心慌意乱的，没看清楚进来的都有哪些人。」

曹粤和李晴的证词听起来倒没有多少出入。

「你陪床期间，离开过病房没有？」冯可欣问得非常细致。

「离开过两回。」曹粤不大好意思地说，「上年纪了，尿频，去过几回卫生间。每次都是去了就回来，前后用不了五分钟。」

「这期间有没有异常情况发生？」冯可欣怕她自主判断哪些情况属于「异常」，哪些属于「正常」，以致漏掉重要线索，又补充了一句，「不管什么情况，只要你能想起来，都可能对案子有帮助。」

曹粤努力回想，有那么一忽似乎想起了什么，眼睛里绽出一线柔柔的光，但很快黯淡下去，她摇摇头：「没有，没发生什么事情。」

冯可欣留意到她的神情变化，鼓励她：「你再想想，不管什么事，多细小的事情都没关系，都说说。」

曹粤缓慢而坚定地说：「没有了。」

胡华华听得入神，精神亢奋：「这个曹粤有鬼，田靓不会是被她害死的吧？这年头，婆媳相杀的故事可不少见。」

肖蔷也赞同他的想法：「婆婆有鬼。」

我说：「冯可欣和你俩的想法差不多，他向沈恕汇报时说，暂时未发现田靓的主治医生和护士有可疑之处，但是曹粤在作证时有一个细微的表情变化，值得深究，毕竟百分之七十的受害人都是被身边的人杀害的。家庭内部的爱恨情仇，不是外人能够看出来的。表面上看，曹粤因田靓的过世感到悲伤，但是谁能肯定她不是装装样子而已呢？」

肖蔷和胡华华都沉浸在我的故事里，似乎已经忘了此行的真正目的，忘了张玉菡。

曹粤从出生到上学，从工作到成家，都未离开过楚原，对她进行背景调查并不困难，冯可欣在下班前就摸清了她的基本情况。

可欣向沈恕汇报：「她是五八年生人，今年刚好六十岁。从楚原市育红学校退休，高级教师职称，教学上兢兢业业，业务娴熟，为人热情开朗，在学校的口碑很好。她丈夫文冲，是原楚原理工大学的教授，三年前因肺癌去世。从那以后，曹粤未再与其他男性交往。

「她生有一儿一女，长子文良宇，就是田靓的丈夫，现任楚原外国语学校副校长，算得上年轻有为。田靓生前在移动公司工作，和文良宇育有一子，今年四岁，入读红黄蓝幼儿园。据熟悉的人说，他们夫妻的感情稳定，互敬互爱，虽然田靓的身体不好，但是文良宇从未嫌弃过她，也从未动过别的心思。」

沈恕的眉头紧锁：「所以看不出这母子二人有杀害田靓的动机？」

「常规的杀人动机无非情、仇、财三种，这母子俩都不沾边，至少从目前掌握的情况来看是这样。」

「曹粤女儿的情况呢？」

「她叫文彤彤，就在田靓出事的省医大第一附属医院上班。」

「哦，」沈恕的眉毛扬起来，「真巧。」

「是啊，文彤彤是儿科护士，田靓去世的当晚她正好值班。不过文彤彤工作的部门在六楼，和田靓的病房隔着两个楼层。」

「文彤彤的个人情况查清楚没有？」

「查过了。她今年二十二岁，未婚，与曹粤同住。楚原医学院护理专业毕业，在省医大第一附属医院工作了近两年。工作细致认真，对待病人热情耐心，是上一年度的最佳新员工。她没有男朋友，下班后极少外出，基本在家陪伴母亲，生活非常简单。她喜欢写作，是市作协会员，曾出版过两本散文集。」

「亲嫂子住院，她当晚又刚好在医院值班，难道没去病房探望过吗？这不合情理。可是曹粤在整个调查过程中都没提到文彤彤的名字，是不是她的那个不经意的表情变化就因为这个？曹粤替她自己或女儿在掩盖什么？」沈恕一边推理，一边用指头有节奏地敲击桌子，嗒嗒嗒嗒地响。

「我也往这方面考虑过，」冯可欣说，「可是找不到曹粤或文彤彤的作案动机。田靓生前身体不好，如果文家想摆脱她，离婚就是了，有必要杀人吗？」

「或者文家和田靓之间有不为人知的宿怨，谁知道呢。」沈恕也挺头疼，「就目前掌握的情况来看，还不够立案条件。你再从外围查查，注意别惊了文家人。如果田靓真的是被人杀害的，那么文良宇主动报案，说明他没有卷入到案件中，他是不知情的。如果曹粤和文彤彤和本案有关联，两人一旦订立攻守同盟，在找突破口就很难了。」
