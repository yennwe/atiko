真实缉毒手记：惊心动魄的缉毒故事

《真实缉毒手记：一个被骗带毒的故事》

夏天，下午，四十多度。我们两个人蹲在路边的草丛里等一辆红色马自达。 雨在头顶悬而未下，蚊子在体表实施放血疗法。 饭前集合哨响起的时候，我们没有等到目标车辆，任务结束。这类协查通报命中率本来就很低。 吃饭的时候，丁卓不太甘心，他似乎对情报来源有着莫名的信任，所以他决定出去把辖区的路重新梳一遍，也许会有新发现。 叫上驾驶员罗K，我们三个人开地方牌照的猎豹车出发。 云南昼夜温差大，出发的时候已经从酷日炎炎变成了春风拂面，这时候下起了暴雨。雨一股一股的，像是电压不足的水泵。 辖区走了半小时，走到我们下午喂蚊子的那条路，由北向南开。本来没抱什么希望，却偏偏在两公里外发现了停在路边的红色马自达，车牌也对得上，车身没有泥，应该是雨前就停在这里的。 但是即使一切特征都符合，搜出毒品的概率也不到五成。这是经验。 我们没有停车，迎面开了过去，然后通知单位把北端的路封起来。 我们掉头，由南向北开，最后停在马自达后面。 罗K没动，我和丁卓下车，准备上去敲车窗，马自达突然启动了。我们赶紧追上去。 马自达可能看到了前面的阻车器，没开多久就停了下来，一个椭圆形的姑娘从车上跑下来。 我们停车的时候，姑娘已经跑到了路边，先是往山下跑，摔倒之后调整姿势，顺着草坪滑了下去，像个调皮的小孩。 虽然部队讲究站如松、行如风，但我们还是没她快，嗖得一下就到山脚了，只留给我我们一个凛冽的背影。 我们顺着背影往下追，距离越来越近，但坦白说，还有一段距离，她这时要是选择跳江，我们很难阻止。 但是她可能是情急之下选错了地方，在江边一个沙坑旁边站定，犹豫了半秒钟，很决绝地跳了下去，像是抱了必死的决心。 那个沙坑面积有几百平米，岸深不到两米，水深五十厘米吧，我实在不知道她为什么跳下去，水中的沙坑像沼泽，越晃越陷越深。她本来就重，落地之后就坐如钟了。 我至今想不通她为什么会往下跳。我问过她，她低头就哭。 然后我们到现场，把她拔了出来。她哭的特别惨，站都站不起来，一直道歉。 然后在她的备胎里发现了六块海洛因。 这个姑娘是湖南人，会计，网上谈了个芒市男朋友，就去芒市约会，约会结束时，男朋友让她帮忙带了点坚果给昆明的朋友。 她顺利完成了任务。 第二次再去看男朋友的时候，她被告知，上次带的坚果里是海洛因。这就像舌吻之后，发现对方是从尿疗协会走出来的一样。 姑娘被人类的各种极端情绪困扰了几天，最终还是答应下水。 她开着马自达，到我们单位附近的时候，由于知道我们单位的位置，就突然不敢前进了，想等夜里再走，等了整整一下午，这也是我喂了半天蚊子也没有等到她的原因。 姑娘第二天就被移交了，后续不详。

《真实缉毒手记：下面是一个盯梢毒贩的过程》

我们有天下午在巡逻，突然有个线人打电话来，说家门口有个外地人在徘徊。虽然不能断定就是毒贩，但总比漫无目的的巡逻要提神，说不定就飞来横货了呢？ 开车过去，人穿便衣，车挂便牌，我们在公路旁边锁定了目标，那人也没干啥，穿着黑衬衫黑裤子，像当地人一样坐在街边抽烟，要不是线人提醒，我们也看不出他是外地人。 我们一看就知道这个人动不得，要在现场把他铲了，身上肯定搜不出任何违禁品，非常干净，比矿泉水倒头上洗的还干净。 因为身上有货的人一般不敢在这种地方长期停留，如果他是毒贩，就一定是接货人，送货人还在送死的路上。 他抽完烟之后，开始往江边走。这就很难搞了，因为江边开阔，四周无人，近距离跟控几乎不可能。 丁卓决定分头行动，我和老狗熊、老狮子三人被留在街上，他和驾驶员开车上山，在高处架起望远镜观察目标。 黑衣人一直在附近徘徊，一会到街上，一会到江边。傍晚的时候，他又走到了江边。坦白说我很着急，我一饿胃就疼，所以无比怀念老左用煳肉做出来的芹菜黑丝。 这时候丁卓在通话器里说，黑衣人开始频繁看手机了，看样子有点急，可能要打电话，让我们想办法接近一点，如果有可能，尽量听清他在电话里说了什么。 这种事只能去一个人，老狮子略胖，趴地上跟个坟包似的，明显不适合潜行，老狗熊倒是瘦，但他正坐在路边自食其火龙果，把半个火龙果对我摇了摇说你看我还没吃完。 那只能我去了，在接近江边的路上有一片杂树林，路虽然不好走，但很隐蔽。我进入树林的时候，丁卓突然下令：低姿匍匐前进，目标往树林这边走了。 低姿匍匐就是趴地上，全身贴地，往前蠕动，那是夏天，衣服很薄，地面上全是树枝石头还有刺，这些都无所谓，最坑的是当地盛产眼镜蛇，运气不好就会被眼镜蛇开光，What 妈的怎么爬？ 丁卓似乎也意识到这个建议不妥，说如果不方便，你就高姿匍匐吧，我现在能看到你，大胆往前爬就是了。 可我又不傻，你望远镜看得再清楚，也没办法望到眼镜啊。 但我好歹入伍时宣过誓，这时候又不能叛离军队，所以只能往前爬，当我爬到树丛深处的时候，丁卓突然说注意注意，他往你的方向来了。 我往左边一滚，滚到了草丛里，这下我估计丁卓也看不到我了。大约半分钟之后，黑衣人的下半身出现在我的视线中，距离我只有五六米的样子，手里拿着手机，选了一小片草浅的地方，用脚踢了踢，然后竟然开始脱裤子大便。 他要是发现我，肯定把我当成变态。我脑补出他瞪着我的样子，估计裤子都来不及提，而我也只能装作目不识丁继续往前爬，等于默认了自己是个变态。 不过他并没有发现我，蹲下之后开始专心蓄力，嗯了几声之后终于拨通电话，这对我来说是个好消息。 电话内容简单说就是问对方到哪了，关键信息一听就是要交接毒品，还约定二十分钟后在上次的厕所交接。说完正事竟然还扯了几句蛋，说什么赚到钱要干嘛干嘛，结尾还互相祝福一下，完全当我不存在，气死我了。 我趴在草丛里，只能感叹命运无常，两个憧憬着福如东海寿比南山的傻子，不幸遇到了我，结局只能是福如东海破船，寿比南山小昙，这就是命。 黑衣人挂掉电话，提上裤子离开，一会儿之后老狗熊汇报说黑衣人出现在街上，我开始跟丁卓汇报情况，丁卓听说好了我知道了，你爬回来吧…… 我还真爬了一截，突然意识到黑衣人已经在街上了，我为什么还要爬？ 街上有两个公共厕所，一个是加油站的厕所，另一个是村里建给游客的。老狗熊和老狮子在这里产生了巨大的分歧，老狗熊认为加油站厕所人少，所以会在加油站交接。但老狮子认为游客厕所有门，隐蔽性更好，所以会在游客厕所交接。 两人为这个还争了几句，丁卓说吵个屁啊，你们俩一人去一个厕所，找个坑蹲下。 我到街上的时候，他们俩估计已经蹲在厕所里了，然后丁卓也开车下来了。丁卓到位之后，黑衣人走进了游客厕所。老狮子此刻正在里面蹲坑，他心里肯定很爽。虽然他和老狗熊分别去一个厕所蹲坑，但毕竟狮屎胜于熊便，老狮子蹲对了。 大约两分钟后，江对岸步行过来一个人，也走进了游客厕所，老狮子蹲在厕所里，两人交接过程没有说话，等他们交接完之后，老狮子发出敲麦信号，大概意思就「铲他们」 抓人过程很简单，先是两人一组，把他们控制住，然后表明身份，拍了拍黑衣人的后腰，发现绑着两块海洛因，接着就塞进了车里，整个过程估计不到三十秒，黑衣人估计没反应过来，上车之后才开始哇哇叫，但无论你怎么哇到最后都是盖木欧瓦。

《真实缉毒手记：今天说一个不为钱的毒贩》

对，就叫阿瓦。 这个人是老左发现的，老左是二级士官，因为负了伤才去后勤班的。 出去买菜的时候，老左在路边小店点了一份米线，就是白水煮熟，上面浇一勺肉酱那种，我们也经常吃。 老左点完餐，把袋子放在凳子上，自己就到附近采购去了。等回来的时候，米线刚好端上来。结果阿瓦把老左的袋子拿到一边，坐下准备开吃。 老左生平最恨这种厚颜无值的行为，两人为那碗米线到底是谁的稍微争了几句。 这种吵架很平常，大家都很饿嘛。但老左成功捕捉到了对方怒不可饿又极力克制的样子，类似于「我要不是赶着过江我今天就弄死你」这类话。 他是真的不了解老左，老左是那种「我不同意你的观点，但我誓死捍卫干掉你的权力」的人，以老左的战斗力，即使腿脚不便，也可以轻松把他按在地上发电，能发很多瓦。 同时老左也是非常敏感的，阿瓦这种又愤怒又克制的情绪，在老左看来很可疑。所以他马上吃了那碗米线，给丁卓打了电话。丁卓当时是我们单位主管缉毒的副职领导。 我们正在打扫营区树叶，被丁卓叫走，换上便衣，开车去往卖菜的集市。必须承认，扫地扫到一半被叫走，那种感觉实在令人窃喜。 老左一直在米线店附近看着那个人，我们到的时候他刚好吃完米线，我们过去，自报了身份，检查他的背包，背包里除了饮料和零食之外，没有什么特别的东西。 把包递给他之后，我一边问话一边在他身上拍，这个动作看上去像是安慰，实际上是一种比较含蓄的搜身，但浑身上下能藏的地方都要拍到，这样一来又很不含蓄，但必须装作很含蓄。 当拍到他大腿内侧的时候，突然碰到一块格格不入的硬物。 这种触感甚至不用看，就知道是什么。带上车，检查一看，果然是块状海洛因。 阿瓦就这样被带了回去，老左继续买菜去了。 回去的路上，阿瓦说「我不是为了钱」，我没想到的是，这句话除了适用于毒贩，我自己成为写作者之后，一度也十分喜欢，但后面往往要加一句「但我也是要吃饭的」 阿瓦说自己打过很多份工，后来觉得自己不能一直打工，就计划先在家看一年书，好好陶冶一下节操，然后再出去工作。 书的确看了不少，每天捧着卡耐基、稻盛和夫这一类作家的书，浑身上下都散发着一股雅不可耐的气息。 读了半个月左右，阿瓦感觉浑身充满了力量，然后他毅然放下书本，走进网吧，把有限的力量投入到了无限的网游中。 那个游戏应该叫《天龙八部》，我没玩过，之所以能记住完全是因为金庸。 阿瓦在游戏里先被中奖信息骗了一次，接着认识了一个云南大哥，大哥装备很好。熟了之后大哥让阿瓦到云南工作，我不记得是以游戏中的道具还是账号作为报酬了。 反正阿瓦同意了。 那天我们抓了阿瓦，半小时之后又在对面抓了他的接货人，是一个黄毛，直接被我们按在了车里。

《真实缉毒手记：今天说一个智力正常的鸭》

李十一是个鸭，后来有一个缅甸华裔富商的女儿看上了他，要和他结婚。 「待到秋来九月八，变成牛粪等花插」，这是每一只鸭的梦想，虽然要娶的人又胖又丑，但至少丑的稳定，总比夜场里不期而至的丑要强很多。 就这样，鸭入赘了。 他很快就发现情况和自己想象的不一样，岳父并不是富商，而是一个毒贩。 他去找岳父核实情况，岳父感叹果然是春江水暖鸭先知，所以就很坦诚地告诉了他，得到确定的答案之后，鸭很生气，反问岳父：你做这种事怎么能不带上我呢？ 从那之后，鸭算是突然暴富。 暴富之后，鸭开始嫌弃家里的老婆，一心想要摆脱那个横看成岭侧要疯的女人。可是岳父在缅甸一手遮天，鸭要是敢在缅甸放肆，岳父能当场把他烤了卷饼吃。 所以，鸭只好回国释放欲望，这对鸭来说轻车熟路。 鸭入行的时候，行业的要求是，一对一的聊天儿，甚至不需要喝酒，聊十分钟就可以让 80 后或 90 后在鸭怀里哭泣。这是入行基础技能，如果你也掌握了，那么恭喜你，你可以入行了。 就这样，鸭在境内找了一个相好的姑娘，性生活开始双系统运行，境内一套，境外一套，这样办起事来很方便。但是，双系统运行的弊病是容易死机。 鸭在帮岳父打理生意的时候，自己会偷偷扣一些下来，交给境内的相好，然后由相好卖出去。 但是，鸭的相好贩卖起来很吃力，因为这一行是需要足够积累的，否则你有货也卖不出去。鸭受制于岳父，也不敢帮她太多，只是给了她一个号码，而她虽然也是个混混，但以前却没碰过这个。 姑娘打通那个号码，说出鸭的名字，对方取得互信之后，达成初步意向。 那天小雨，我们驻地附近的集市上人来人往，摊贩占满了街道两旁，很多妇女只好把水果放到篮子里，挎着篮子做流动摊贩。 集市是我们重点关注的地方，所以每到赶集那天，我们都会过来看看，那天虽然下雨，但也照常过去。 我们盯上了一个挎篮子卖香蕉的年轻女人，她和普通摊贩不同，比其他流动摊贩都要年轻的多，肤色气质都不像当地的劳动者。别人都是到处吆喝叫卖，但她却一声不叫，只顾挎着篮子四处张望，似乎在找什么人。 我们上前去，说明身份，检查她篮子里的香蕉，发现香蕉表面有裂痕与胶水粘合的痕迹，剥开，里面的果肉已经没有了，只有塑料袋包裹的毒品。 我们不动声色地准备将女子带走，尽可能不让周围人发现，我们还想抓接货人。 但事情在这里起了变化，五六个穿着紧身裤，顶着非主流发型的年轻人举着棍棒砍刀高喊着放人，就冲过来了。 我现在回忆起他们一个个热血沸腾，壮怀激烈，嗷嗷待捕冲过来的样子，还是会觉得很刺激，中学毕业以后我已经很久没看到这样的打扮了。 我们人数和他们差不多，他们个个瘦得像甘蔗，自然不会对我们造成威胁，可以说我们把一只手绑裤腰带上都能拿下他们，因为开枪只要一只手就行了。 但我们都有两只手，所以没有开枪，街上人多，影响太不好了。所以就靠着勤劳的双手按住他们，遗憾的是，这么一闹，接货人没能抓住。 姑娘被抓之后，主动配合工作，所以很快把鸭骗回国内，双双落网。 至于鸭的结局……数罪并罚，盖木欧瓦。

《真实缉毒手记：以下是一个法盲抗法的故事》

真实缉毒手记：IT男贩毒漏洞百出，人赃并获

那天是午饭时间，但饭太难吃了，我和老狗熊活活被老左逼成了逃饭，逃回宿舍去吃泡面。 但只要轮到你值日，你可以不吃饭，等别人吃完之后，你必须要洗全桌人的碗。 那天老狗熊值日，我们俩一起出来，老狗熊去洗碗，我去厕所。 路上我看到丁卓在远处操场上，一个过肩摔把另一个新兵摔了两百七十度，然后把人按在地上。 丁卓那按兵不动的姿势像是要掐死对方，我还以为跟谁决斗呢，赶紧跑过去，走近了才发现，原来在教新兵摔擒。 作为老兵的经验告诉我，这种传授武功的场面不能多呆。 我转身想走，但仍然迟了一步，被丁卓叫了回来，要我配合一下，说是要给新兵演示。简单说就是他要打给人家看，而我就是那个被打的。 我告诉丁卓可以打旁边的沙袋。 丁卓说沙袋还是活的好。 我闭眼感受了一下，确定自己现在的状态很容易被打爆，就是打成失禁状态，所以只能假装听不到跑了。 在我迅速远去的时候，单位通讯员却迎面跑来，然后以一个沙包的姿势朝丁卓跑去。 那我就不着急逃了，我往回跑准备看武打片。 他跑到丁卓面前，说了一下情况，简单说就是一男一女两个人出现在我们的区域，看上去行色匆匆的，像外地人，可疑。 我们集合，朝目的地出发，走得很快，目标位置在山背面，那一男一女也没走，就坐在原地休息。 女人背对我们坐在地上，男人悬浮在半空中，能看到侧脸，倒是不太像贩毒的人，没有罪犯气质，而且愁眉苦脸的，看上去倒像个文艺青年。人往石头上一站，有种郁树临风的气质。 然后我们就围上去，先是亮出自己的证件，然后检查了他们的证件，男人湖南的，女人四川的，俩人说是来徒步旅行的。 那我们当然不信，当地四十多度的气温，不带这么玩的。问他是做什么的，他好像说是软件工程师，我当时对这个职业完全无知，我的理解就是网管、修电脑的、或者高级一点做网页的。毕竟我自己当时也只是熟练操作QQ的水平。 这样的问话是有技巧的，比如他说他是软件工程师，你得问他软件行业的事情，才能确认他的身份有没有疑点。要是现在问，我可能会装作内行和他讨论下 Python 和 Java 的优劣，但那时候只知道问他会不会换电脑硬盘，或者使几台电脑联机，因为当时我们正在为不能连局域网打暗黑而苦恼。 得到的答案是都会，问话的确难不住他，还搞得我们很尴尬，于是我们开始搜男人，没有什么发现，行李也没啥发现。最后在女人的背包里发现一大包糖果。 丁卓说你这个糖果我们打开两颗看看行吧？正常人一般都会同意。认真点的人，会说你打开就得赔我钱。 但那个女人特别耿直地说：这个你们不能查。 问她为什么不能查，她也不说话。 丁卓说：你也知道我们是干什么的，既然你都这么说了，无论如何我都要查了。 女人想来夺包，被老狗熊拦住，丁卓开始随机剥糖果，大概就是每个颜色挑几颗出来，一共大概剥了五六颗，在紫色糖果里发现了海洛因。 俩人现场被上了铐子，带回单位的路上，丁卓开始问话，这时候发生一点小意外。 本来整个过程中，男人都没怎么说话，就是那种停智不前的样子，但上车之后突然变了个人，有了一种智咤风云的感觉。 他开始诉说自己的悲惨身世，说自己得了癌症，带货是为了治病。一边说还一边让女人给自己证明，可能是男人的即兴发挥，俩人没有提前串供，所以配合起来就有点磕巴。 男人说自己得癌症的时候，用下巴指了指女人，说：不信你们问她。 女人似乎 get 到了男人的点，而且非常紧张，说：对，我也得了癌症，我们俩都是为了治病。 俩人越说越惨，就差唱癌的路上只有我和你了。 后来审讯细节我没有参加，内勤做完笔录之后，我拿着路由器去找男人问了几个问题，从那以后就可以联机打暗黑了。 具体数量我记不清了，只记得货是两人从境外背回来的，第二天一早移交，数罪并罚，盖木欧瓦。
